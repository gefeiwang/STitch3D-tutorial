Installation 
============

Installation
------------
STitch3D package is freely available at https://github.com/YangLabHKUST/STitch3D

.. code-block:: python

   git clone https://github.com/YangLabHKUST/STitch3D.git
   cd STitch3D
   conda env update --f environment.yml
   conda activate stitch3d

.. code-block:: python

   import STitch3D
